Como usar

  • Busca expedientes desde la pestana Buscar por numero, caratula o partes
  • Tambien podes navegar en plataforma.justiciasalta.gov.ar y la extension detecta el expediente
  • Usa "Monitorear expediente" para recibir notificaciones de novedades

Seguimiento

  • Usa "Monitorear expediente" para seguir cambios
  • La extension verifica automaticamente cada 15 minutos
  • Si hay movimientos nuevos, recibis una notificacion de Chrome
  • Los expedientes con novedades se resaltan en la pestana Seguimiento

Importacion masiva

  • Prepara un Excel (.xlsx) o CSV con una columna de numeros de expediente
  • La extension buscara cada expediente en la API publica
  • Podes exportar los resultados y seguir todos los encontrados

Descarga

  • Desde un expediente, usa "Descargar todo (ZIP)" para obtener PDFs
  • Cada actuacion se descarga como PDF si esta disponible
  • Se incluye un informe general automatico en el ZIP

Informe con IA

  • Obtene una API key gratis en aistudio.google.com
  • Pegala en la configuracion de abajo
  • Desde un expediente, usa "Informe IA" para obtener un analisis inteligente
